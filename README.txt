# Crypto donation alert script for OBS (USDT TRC20)
How to install it:
 - download files: `index.html` & `coin.mp3` (you can change it)
 - open index.html and change line from `"put_here_your_wallet"` to your real USDT TRC20 address and save the file
 - add new source "Browser" in OBS
 - choose index.html filepath like: `C:\Users\CryptoBroUser\Downloads\index.html` as a file/url
 - done!
